Descrição: Game de adivinhação de número com Javascript.



Implementado:



* Centralização de variáveis, evitando magic numbers (números soltos que precisam ser alterados em mais de um lugar);
* Geração automática de um número secreto pseudo-aleatório para cada vez que o sistema é iniciado;
* Contagem de tentativas realizadas;
* Aplicação de operação ternária para retorno correto do plural e singular, reduzindo grande quantidade de linhas;



Oportunidades para upgrade:



* Haver um número máximo de tentativas;
* Retorno para quando o valor de entrada for vazio;
* Existir um retorno diferente quando o valor de entrada for acima do que é estipulado no número máximo;
* Alterar essas possibilidades de retorno também no texto que é apresentado no html, não só como alert;
